echo # Weather Data Visualizer

This mini-project was created for the Programming for Problem Solving using Python course. The goal was to clean, analyze, and visualize weather data.

## Project Contents
- weather_dataset.csv
- cleaned_weather.csv
- Python script / Jupyter notebook
- PNG plots
- summary_report.md

## What the Project Does
- Loads and cleans the weather dataset
- Groups data by month
- Calculates statistics using NumPy
- Creates multiple visualizations
- Saves all plots as PNG images

## How to Run
pip install pandas numpy matplotlib
python weather_project.py

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
